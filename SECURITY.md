# Security Policy

We take security seriously. If you discover a security vulnerability, we appreciate your help in reporting it responsibly.

## Reporting a Vulnerability

Please do **NOT** open a public GitHub Issue for security bugs or vulnerabilities. Instead, report them by emailing `security@cbtpro.org`. 

We will acknowledge receipt of your report within 48 hours and work with you to patch the issue before making a public disclosure.

## Supported Versions

We actively provide security patches to the following versions of CBTPro:

| Version | Supported |
| :--- | :--- |
| v1.0.x |  Yes |
| < v1.0.0 | No |
